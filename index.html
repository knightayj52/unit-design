브라우저에서 Gemini 직접 호출이 되는지 확인

서버(앱스크립트) 없이, 이 페이지의 자바스크립트가 Gemini API를 곧장 부릅니다. 여기서 응답이 돌아오면, 앱 전체를 GitHub Pages로 옮길 수 있다는 뜻입니다.

1. 내 Gemini API 키

키는 이 브라우저에만 저장되고 어디로도 전송되지 않습니다(저장 위치: localStorage). 키가 없으면 Google AI Studio에서 무료로 발급합니다.

2. 테스트 실행

연결 테스트는 한 줄 응답으로 호출 자체가 되는지 봅니다. JSON 후보 테스트는 영쌤 앱의 실제 패턴(엄격한 JSON 출력 → 파싱 → 카드)이 정적 환경에서도 도는지 확인합니다.
